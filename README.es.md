## 📖 ¿Qué es MyZubster App?

La aplicación Android oficial para [MyZubster](https://github.com/DanielIoni-creator/MyZubsterAPP) – una pasarela de pago Monero autoalojada y marketplace de habilidades.

## 🔗 Integración con MyZubster

| Backend | URL | Rol |
|---------|-----|-----|
| **MyZubster Core** | `http://localhost:3000` | Pasarela de pagos Monero |
| **MyZubster Marketplace** | `http://localhost:4000` | API de habilidades y pedidos |
