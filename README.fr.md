## 📖 Qu'est-ce que MyZubster App ?

L'application Android officielle pour [MyZubster](https://github.com/DanielIoni-creator/MyZubsterAPP) – une passerelle de paiement Monero auto-hébergée et un marché de compétences.

## 🔗 Intégration avec MyZubster

| Backend | URL | Rôle |
|---------|-----|------|
| **MyZubster Core** | `http://localhost:3000` | Passerelle de paiements Monero |
| **MyZubster Marketplace** | `http://localhost:4000` | API de compétences et commandes |
