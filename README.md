\## 📖 About



This is the official \*\*Android mobile app\*\* for \[MyZubster](https://github.com/DanielIoni-creator/MyZubsterAPP) – a self-hosted Monero payment gateway and skills marketplace.



\---



\## 🚀 Status



🚧 \*\*Under development\*\* – coming soon!



This repository will contain the Android app for MyZubster, built with Kotlin and Jetpack Compose.



\---



\## 🔗 Integration with MyZubster



| Backend | URL | Role |

|---------|-----|------|

| \*\*MyZubster Core\*\* | `http://localhost:3000` | Payment gateway (Monero) |

| \*\*MyZubster Marketplace\*\* | `http://localhost:4000` | Skills and orders API |
