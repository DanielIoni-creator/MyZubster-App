## 📖 Cos'è MyZubster App?

L'app Android ufficiale per [MyZubster](https://github.com/DanielIoni-creator/MyZubsterAPP) – un gateway di pagamento Monero self-hosted e marketplace di competenze.

## 🔗 Integrazione con MyZubster

| Backend | URL | Ruolo |
|---------|-----|-------|
| **MyZubster Core** | `http://localhost:3000` | Gateway pagamenti Monero |
| **MyZubster Marketplace** | `http://localhost:4000` | API competenze e ordini |
